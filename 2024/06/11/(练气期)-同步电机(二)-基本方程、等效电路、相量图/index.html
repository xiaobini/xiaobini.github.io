同步电机-基本方程、等效电路、相量图（二）

隐极同步电机

隐极同步发电机

根据上节的隐极同步电机中提到的电磁关系，在隐极同步电机带载的情况下（如图所示）：

转子：$I_f->F_f->Φ_f->E_0$​

定子：$U->I_a->F_a->Φ_a->E_a$

​ $|->Φ_σ->E_σ = -jX_σI_a$

​ $|->-I_ar_a$

在图中除了定子产生的电枢磁场$E_a$以外，在线圈上还存在一个$r_a$的阻抗。除此之外还存在一个不通过气隙的漏磁通$E_σ$

在磁路不饱和的时候$E_a∝Φ_a∝F_a∝I_a$在此基础上，不考虑铁耗的情况下$Φ_a$与$I_a$同向位。

由于$E_a$滞后于$Φ_a90°$，所以在$E_a$的表达式可以写成：

​ $E_a = -jX_aI_a$

其中$X_a$表示电抗（电枢反应电抗）反应的是磁通在气隙中磁链的情况。j是旋转因子

$X_a$可以表示为：$X_a = ω_1L_a = 2pif_1(N_1K_{w1})^2Λ_a$​

在式中可以看出电枢感应电抗与磁导有关，气隙越大，磁导$Λ_a$越小则电抗越小。反之气隙越小，磁导$Λ_a$越大则电抗越大

同理漏电势$E_σ = -jX_σI_a$​

则在磁通不饱和情况下：$U = E_0 + E_a + E_σ - I_ar_a$​将电抗表达式带入得到

​ $E_0 = U - E_a - E_σ + I_ar_a = U+jX_aI_a + jX_σI_a + I_ar_a = U+jX_tI_a + I_ar_a$ $(2-1)$

其中$X_t = X_a + X_σ$表示隐极同步电机的同步电抗，反应的是电枢反应磁通与漏磁通经过磁路的情况，当磁路不饱和的时候$X_t$是一个常数。

隐极同步电动机

隐极同步电动机与发电机之间的区别在于$I_a$​的方向，在电动机中由转子产生的磁势给到外面的负载。而电动机是外部输入三相电，所以电动机变成一个负载。

电压平衡方程为：$E_0 = U - I_ar_a - jX_tI_a$​

常用标幺值计算

标幺值可以说是参数的另外一种表现形式：

标幺值计算公式为：$标幺值= 实际值/基值$

对于隐极发电机，若已知$U、I、cosψ$

功率：功率$P_1$选择使用额定功率$P_n$作为基值。$P = P_1/P_n$

电压：电压$U_0$使用相电压$U_{фb}$做为基值。$U = U_0/U_{фb}$

电流：电流$I_0$使用相电流$I_{фb}$做为基值。$I = I_0/I_{фb}$

阻抗/电抗：阻抗的基质使用的是$Z_b =U_{фb}/I_{фb}$。$Z = Z_k/Z_b$

凸极同步电机

凸极电机的分析方法与隐极电机的分析方法相似，但是由于凸极电机的气隙磁场不均匀，所以在分析上与隐极电机有些出处。

凸极同步发电机

凸极电机的电磁关系如下所示：

转子： $I_f->F_f->Φ_0->E_0$

定子：$U->I_a->F_a->F_{ad}->Φ_{ad}->E_{ad}$

​ $->F_{aq}->Φ_{aq}->E_{aq}$

​ $->Φ_{σ}->E_σ = -jX_σI_σ$

​ $->-I_{a}r_a$

**当磁路不计饱和的时，直轴电枢电势$E_{ad}$表示为$E_{ad} ∝Φ_{ad}∝F_{ad}∝I_d$，交轴电枢电势$E_{aq}$表示为$E_{aq} ∝Φ_{aq}∝F_{aq}∝I_q$**。

由于$E$滞后于$Φ90°$，则$E_{ad}、E_{aq}$可以表示为：

​ $E_{ad} = -jX_{ad}I_{ad}$

​ $E_{aq} = -jX_{aq}I_{aq}$

$X_{ad}、X_{aq}$分别表示凸极电机直轴和交轴的电抗，可以表示为：

​ $X_{ad} = ω_1L_ad = 2pif_1(N_1k_{w1})^2Λ_{ad}$

​ $X_{aq} = ω_1L_aq = 2pif_1(N_1k_{w1})^2Λ_{aq}$

其中$Λ_{ad}、Λ_{aq}$与对应的气隙大小成反比。由于$Λ_{ad} > Λ_{aq}$，所以$X_{ad} > X_{aq}$。在隐极同步电机中，由于$Λ_{ad} = Λ_{aq}$，所以$X_{ad} = X_{aq}$。所以隐极电机可以看成是特殊的凸极电机。

根据KVL公式可得：

​ $E_0+E_{ad}+E_{aq}+E_σ-I_ar_a = U$

可以推出：

​ $E_0 = U + JX_{ad}I_{ad} + JX_{aq}I_{aq}+JX_σ(I_{ad}+I_{aq})+I_ar_a = U+I_ar_a+jX_dI_d+jX_qI_q$

其中$X_d = X_σ+X_{ad}$称之为直轴同步电抗，$X_q = X_σ + X_{aq}$称之为交轴同步电抗。

无论哪种电抗都与对应的电枢电流相对应，对于任意一相的定子绕组电抗表示的是三相合成磁场在一相中感应电势的能力，而不是只有这一相绕组产生的磁场。

凸极同步电动机

与隐极同步电动机一样，凸极同步电机只改变基本方程即可。
$E_0 = U-I_ar_a-jX_dI_d-jX_qI_q$

即

​ $U = E_0+I_ar_a+jX_aI_d+jX_q+I_q$
