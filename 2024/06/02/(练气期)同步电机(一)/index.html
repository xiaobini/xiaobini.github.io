同步电机的基础知识

同步电机指的是转子的旋转速度与定子产生的磁场的旋转速度相同共同转动（同步速）。

同步速的转速：$n_1 = 60f_1/p$,其中p指的是极对数，f指的是频率。

一些基本概念

电枢反应：定子绕组产生的电枢磁场。

励磁电流：同步电机转子中通入直流电流所产生的电流。

电磁基本关系及常用公式

磁感应强度B：用来描述磁场的大小

P点受到长度为dl磁场的大小为$dB = kIdla/R^2$,如果说有的长度磁场都叠加在P点时则P点受到的磁场为：$B = ∫dB$

磁通Φ：磁通表示的是在空间内在一定空间面积内磁场经过的磁力线总数

空间中磁通计算为：$Φ = ∫_sBds$,如果是均匀磁场，B与S垂直的情况下公式可以变成$Φ = BS$

磁感应强度H：表示的是电流产生的磁场大小，表征磁场性质的另外一种基本物理量。

磁通与磁场强度的比值表示的是磁性材料的导磁能力：

​ $B = μH$

μ指的是磁导率。

安倍环路定理：描述的是导体电流与导体产生的磁场之间的关系，是电生磁的基本定律。

​ $∫_LHdl = Σi_k$

如果磁力线由N匝电流产生并且磁场强度H处处相等，则公式可以变为

​ $HL = Ni = F$

F称之为磁势，$F=NI$

法拉第电磁感应定律：磁生电的基本定律，描述的是感应电势与磁场之间基本关系

​ $e = -NdΦ/dt = -dψ/dt$​

N为匝数，ψ为磁链（表示的是N匝线圈形成的总磁通$ψ=NΦ$）。

由楞次定律可知，当磁场减小或增大的时候，为了防止磁场的突变会产生感应电动势/反电动势。这种变化与磁场的方向相反，所以在产生感应电动势的时候需要加上一个负号

磁路中的欧姆定律

磁路：磁通经过的路径。

假设磁导率为μ，铁芯的横截面积S均匀，N匝线圈并且每匝线圈通过的电流为i,磁路的平均长度为l，则

[!IMPORTANT]

$F = NI=Hl = Bl/μ = Φl/μS = ΦR_m = Φ/Λ_m$​

这是一个比较重要的等式，其中$l/μS$是一个常数，所以可以当成是磁阻，$Λ_m$是磁导，$Λ_m = 1/R_m$,反映的是磁路的导磁能力。

我们可以类比我们学过的电路，在这里，我们可以把磁通Φ看成是电路中的电流，F表示的是电路中的电动势，磁阻$R_m$表示的是电阻。磁感应强度H为电压。

那么可以得出磁路中的基本定律：

磁路基尔霍夫第一定律：描述的是磁路中的总磁通为0，即∑Φ = 0，类比KCL

磁路基尔霍夫第二定律：描述的是在很截面积内磁感应强度为0，即$∫BdS = 0$，类比KVL

同步电机的结构

隐极结构与凸极结构的不同点

隐极电机与凸极结构的主要不同主要是转子与定子之间的气隙不同，由于气隙的不同就会导致磁阻的不同。进而会影响到磁势和磁通。

隐极电机主要用于汽轮发动机，凸极电机主要运用在水轮发电机。

在电机的定子中通入AC电流，所产生的磁场称之为电枢磁场，这样子的反应称之为电枢反应。

同步电机空载情况

电机空载：电机空载的定义主要指的是电机转子在使用原动机提供转子的同步转速。

电机空载的时候，对转子励磁的绕组通入直流电，磁势定子绕组开路（$I_a = 0$这里指的是一相电流，三相电流都为0）。

如果直流励磁的电流位$I_f$,则转子励磁磁势为$F_f=N_f*I_f$

在分析同步电机的时候一般ABC三相只选择其中的一项进行分析，其它两项也是同样的道理。在这里我们选择A相绕组进行分析。

由于在A相定子中产生的感应电势$E_0 = -j4.44F_1N_1K_WΦ_0$,并且同步电机的同步转速为$n_1 = 60f_1/p$（其中$f_1$为感应电势频率，p为电机的极对数）可知，改变转子励磁电流$I_f$会改变磁通的大小，进而会改变感应电势$E_0$的大小。

由$E_0 = -j4.44F_1N_1K_WΦ_0$可知**$E_0$滞后与磁通$Φ_0$90度**。

-j 表示的是滞后90度

在同步电机中，我们通常将转子与定子之间磁通最大时候转子的轴线称之为d轴（直轴），与d轴垂直的轴线称之为q轴（交轴）。

转子转到d轴的时候，磁势转子与定子之间的气隙最小，所以磁路中的磁阻最小，根据$F = ΦR_m$,磁通$Φ = F/R_m$

所以在d轴的时候磁通最大，同理在q轴的时候转子与定子之间的气隙达到最大值，所以磁通最小。

假设电机当前运行状态到达上述电机运行的状态，那么磁势同步电机空载时空向量图可以表示为：

根据相量图可以知道，在当前电机的运行状态下，感应电动势$E_0$最大，但是此时磁通最小（应为电势滞后与磁通90度，那么磁通在q轴上的投影为0，所以磁通为最小值。）

同步电机带载

同步电机带载的时候，定子绕组通入电流$I_a$从而在定子上产生电枢磁势$F_a$以及磁场$B_a$，与转子的磁势和磁场以同步速旋转（相对静止）。定子产生的磁势和磁场与转子励磁产生的磁势和磁场在空隙中合成气隙磁势（$F_σ$）和磁场($B_σ$)。

电枢反应：指的是电枢磁场对主磁场的影响

为了方便计算和画图以下的说明和计算都是建立在磁通不饱和的情况下说明，因为只有在不饱和区磁化曲线是直线，满足叠加定理

由于在同步电机中，励磁磁势与电枢磁势之间以同步速旋转，所以在气隙中合成的磁势为

​ $F_σ = F_a+F_f$

在同步电机中，气隙磁势是电机产生扭矩的重要参数，而气隙磁势的的产生与电枢磁势和转子磁势之间位置有关，也就是$E_0$与$I_a$​

的位置有关，一般将$E_0$与$I_a$之间的夹角称之为内功率因数角ψ。接下来我们就会讨论$E_0$与$I_a$​不同角度对气隙磁势产生的影响。以下的讨论都是讨论ABC相绕组中的其中一组，以及在电机当前如图暂态的情况（其他情况都同理）。

$I_a$滞后$E_0$任意$ψ$时

当$I_a$滞后$E_0$任意$ψ$时的相量图如图所示：

如果电枢磁势此时不在d轴或q轴上，就不能直接按照前面的方法直接合成气隙磁势，因为随着内功率因素角的不同，会导致电枢磁势发生变化。在隐极同步电电机而言由于气隙时均匀的，所以磁导不会发生变化。但是对于凸极同步电机而言，气隙是不均匀的，磁导会发生变化，从而气隙磁势也会发生相应的变化，所以需要做另外的分析。

在这里可以使用分解电枢磁势$F_a$的方式将$F_a$分别映射到q轴和d轴上，分别称为$F_{aq}$与$F_{ad}$。

有相量图可以得出如下等式

​ $F_a = F_{aq}+F_{ad}$

其中：

​ $F_{ad} = F_a*sinψ$

​ $F_{aq} = F_a*cosψ$

上述将$F_a$分解成$F_{ad}$与$F_{aq}$的方法称之为双反应理论，由于我们分解之后得到的气隙磁势就可以按照前面所述的方法进行分析。

由此我们可以得出：

当$F_{aq}$与$F_f$合成磁势的时候是交轴电枢反应，当$F_{aq}$与$F_f$合成磁势的时候是直轴去磁电枢反应。但是是助磁还是去磁还需要具体情况进行 分析，如果$0°<ψ<90°$则为去磁，但是如果$90°<ψ<180°$则为助磁。

除了上述可以将电枢电势分解在分析以外，同理我们也可以将电枢电流进行分解在分析。

电枢电流分解的时空相量图如图所示：

有电流产生的磁势跟电流同方向，所以分解结果跟磁势分解结果一致。

此时可以得出结论如下：

​ $I_a = I_{aq} + I_{ad}$

那么

​ $I_{ad} = I_asinψ$

​ $I_{aq} = I_acosψ$

注意：运用双反应的前提的是磁势与磁通是线性关系，也就是说磁通不饱和的情况。
